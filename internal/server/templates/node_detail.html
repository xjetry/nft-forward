节点安装命令

在目标节点（已安装 nftables，root 用户）执行下列命令；将 nft-agent 二进制提前传到 /usr/local/sbin/：

install -m 0755 -d /etc/nft-forward /var/lib/nft-forward
echo '{{.Node.Secret}}' > /etc/nft-forward/agent.token
chmod 600 /etc/nft-forward/agent.token

cat >/etc/systemd/system/nft-agent.service <<'EOF'
[Unit]
Description=nft-forward agent
After=network-online.target nftables.service
Wants=network-online.target

[Service]
ExecStart=/usr/local/sbin/nft-agent --listen :{{port .Node.Address}} --token-file /etc/nft-forward/agent.token
Restart=on-failure

[Install]
WantedBy=multi-user.target
EOF
systemctl daemon-reload
systemctl enable --now nft-agent.service

完成后回到面板，点击「重新同步」即可验证连通性。
