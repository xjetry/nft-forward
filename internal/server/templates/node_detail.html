节点安装命令

在目标节点（已安装 nftables，root 用户）执行下列命令。把 https://YOUR-PANEL 换成 agent 可达到的面板 URL（反代后的公网域名或内网地址）：

curl -fsSL https://raw.githubusercontent.com/xjetry/nft-forward/main/install.sh -o install.sh
sudo bash install.sh agent \
  --panel-url https://YOUR-PANEL \
  --token {{.Node.Secret}}

install.sh 会装好 daemon、把 token 写入 /etc/nft-forward/panel.token、并把 daemon unit 配成 --connect wss://YOUR-PANEL/v1/agents。无需在节点本机开放任何端口。
